Debug the following problem: $ARGUMENTS

Spawn the **Debug** agent to find and fix the root cause.

Debug process:
1. Understand the exact failure — not just the symptom
2. Form hypotheses ordered by likelihood
3. Binary search to the root cause — eliminate, don't guess
4. Verify the root cause fully explains the failure
5. Fix only the root cause — nothing else
6. Verify the fix resolves the issue without regressions

Output: a Debug Report with root cause, fix applied, verification, and anything to watch.

Never patch a symptom. Find the cause.
