# WHOOP Data

Daily logs pulled from the WHOOP API. Each file contains recovery, strain, sleep, and HRV data.

Format: `YYYY-MM-DD.md`

Claude uses this data to advise on:
- **High recovery (67%+):** Go hard — match play, intense drills, conditioning
- **Moderate recovery (34-66%):** Technique work, light hitting, strategy sessions
- **Low recovery (below 34%):** Rest or very light movement only

These thresholds are starting points. We'll adjust them based on how Colby actually feels vs what the data says.
